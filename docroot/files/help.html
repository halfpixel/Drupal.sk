Cenus is unique template to show off your portfolio/projects. It has modern & minimalist style. Powered by custom jquery. Use it to attract your client.

Installation

  1. Extract Cenus_D7_V01.zip file into any folder, then upload all files to your web server or localhost. If you use web server then copy all the files into root. For install locally, you need download localhost software first. You can install xampp, then copy all the files to C:\Program Files\xampp\htdocs\cenus\
  2. Open address from browser
  3. You will see Drupal installation page. Select Cenus Profile then click Save & Continue
  4. Choose language. then click Save & Continue
  5. Type your database database name, username and password then click Save & Continue
  6. Fill Site Information, Site Maintenance Account And Server Settings then click Save & Continue
  7. Done! Your site is ready to use!

[GENERAL] LOGIN ADMIN & CHANGE PASSWORD

  1. Open your site login page at www.yoursitename.com/?q=user
  2. Login with your preferred username and password when you install for the first time.
  3. Select the Edit tab/link
  4. Type your current password and new password
  5. Confirm your password with retype it ones more.
  6. Click Save.

[GENERAL] Change Logo

Change Main Logo
  1. On the Administration Menu that runs across the top of the page, select Appearance â†’Settings.
  2. Go to the Logo Image settings section
  3. Uncheck Use the default logo checkbox.
  4. In the Upload Logo Image field select your preferred image file by click Choose File.
  5. Click Save Configuration to save settings.
To Edit pavicon:
  1. Similiar with edit logo.
  2. Uncheck Use the default shortcut icon.
  3. In the Upload Logo Image field select your preferred image file by click Choose File.

[GENERAL] Change Theme

Change Theme
  1. On the Administration Menu that runs across the top of the page, select Appearance
  2. Select one of our themes and click Set default.

Portfolio

Create new portfolio content
  1. On the Administration Menu that runs across the top of the page, click Add content and then click Portfolio
  2. Type the title for new blog in the Title Field.
  3. In the Description field enter your new Portfoliocontent.
  4. In the Picture field select your preferred image file. This field is required so you have to add your image. The image will be resize in the propper size.
  5. In the PopupPicture field select your preferred image file. This field is required so you have to add your image. The image will be resize in the propper size.
  6. Scroll to the bottom of the page.
  7. Click the Save button to save your new portfolio.
Edit portfolio content
  1. On the Administration Menu that runs across the top of the page, click Find content
  2. Select Portfolio type and click Filter button.
  3. The portfolio content list wil show up.
  4. Click edit link. The page now opens in edit mode.
  5. Once you done then click Save.

BLOG

CREATE BLOG ENTRY
  1. On the Administration Menu that runs across the top of the page, click Add content and then click Blog entry
  2. Type the title for new blog in the Title Field.
  3. In the Body field enter your new blog content. You can split summary by click on the Split Summary at cursor button and unchek Show summary in full view checkbox. Note that this summary will show in different style.
  4. In the picture field select your preferred image file. This field is required so you have to add your image. The image will be resize in the propper size.
  5. Scroll to the bottom of the page.
  6. Click the Save button to save your new page.
EDIT BLOG ENTRY
  1. Navigate to the blog page you wish to edit. You'll see two links at the top of the page: View & Edit.
  2. Click on Edit. The page now opens in edit mode.
  3. In the Body field, make changes to your content.
  4. Scroll to the bottom of the page.
  5. Click the Save button to save your page.

Custom Page

Create new page
  1. On the Administration Menu that runs across the top of the page, click Add content and then click Basic page
  2. Type the title for new page in the Title Field.
  3. In the Body field enter your new page content.
  4. If you want to add an image file go to the file attachments and upload an image file.
  5. After image is uploaded then copy the url to the image file and paste it to the body field within the HTML image tag .
  6. If you want to attach this page to the primary menu which is show on the left side, then in the Menu Settings section enter the Menu Link Title and select Main menu in the Parent Item field.
  7. Click the Save button to save your new page.

Edit page content
  1. Navigate to the page you wish to edit. You'll see two links at the top of the page: View & Edit.
  2. Click on Edit. The page now opens in edit mode.
  3. In the Body field, make changes to your content.
  4. Scroll to the bottom of the page.
  5. Click the Save button to save your page.

Header & Footer

Edit header & Footer
  1. Login as admin

  2. To edit block header content move your mouse over the header area and click the icon on the top right and then click Configure Block link

  3. To edit block footer content move your mouse over the footer area and click the icon on the top right and then click Configure Block link
